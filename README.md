## Requirements

* NodeJS V20+
* FFMPEG
* Imagemagick
* NodeJS basic knowledges (dont be an idiot)

#### Setup bot on your Machine
```bash
$ git clone https://github.com/watson-xd-2/WATSON-XD-BOT 
$ cd WATSON-XD-BOT 
$ npm i
```

---------

## Run

```bash
npm start
```
